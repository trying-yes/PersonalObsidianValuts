#  链路层（Data Link Layer）协议体系全景

链路层负责：

> 在同一链路（本地网络）中传输数据帧（Frame），并控制访问媒介的方式。

---

## 1️ 常用链路层协议（最主流）

| 协议 / 标准                                 | 传输介质     | 场景         | MAC 地址 | 特点           |
| --------------------------------------- | -------- | ---------- | ------ | ------------ |
| **Ethernet (以太网)**                      | 双绞线 / 光纤 | 局域网 LAN    | ✅ 有    | 最广泛使用的有线网络标准 |
| **Wi-Fi (IEEE 802.11)**                 | 无线电波     | 无线局域网 WLAN | ✅ 有    | 家庭、办公室无线接入   |
| **PPP (Point-to-Point Protocol)**       | 串行线路     | 拨号上网、VPN   | ❌ 无    | 简单点对点封装      |
| **HDLC (High-Level Data Link Control)** | 串行线路     | 路由器互联、工业通信 | ❌ 无    | 面向比特的点对点链路   |
| **Frame Relay**                         | 电信专线     | 广域网 WAN    | ❌ 无    | 面向帧的虚电路服务    |

---

## 2️ 广域网 / 电信网络链路协议

|协议 / 标准|场景|特点|
|---|---|---|
|**ATM (Asynchronous Transfer Mode)**|电信骨干网|面向连接，小固定单元（53字节）|
|**SONET / SDH**|光纤传输主干|高速同步光网络|
|**X.25**|早期分组交换网络|可靠但低效（1970s）|
|**FDDI**|光纤局域网|环形拓扑，已淘汰|
|**Token Ring**|IBM 环网|使用令牌避免冲突|
|**ARCNET**|早期局域网|工业中仍有遗留使用|

---

## 3️ 无线与蜂窝通信链路层

|协议 / 技术|场景|特点|对应代际|
|---|---|---|---|
|**Wi-Fi (802.11)**|局域无线|无线局域网，家用/企业|-|
|**LTE (Long Term Evolution)**|蜂窝网络|4G 标准，全 IP 化链路|4G|
|**LTE-Advanced / LTE-A Pro**|蜂窝网络|LTE 增强版|4G+|
|**5G NR (New Radio)**|蜂窝网络|5G 链路层标准|5G|
|**EDGE / UMTS / WCDMA**|早期蜂窝网络|2G–3G 链路层技术|2G–3G|

这些协议属于**无线物理层 + 链路层的组合实现**，在结构上和 Wi-Fi 地位相同，只是场景不同（广域 vs 局域）。

---

## 4️ 工业与专用场景链路层协议

|协议 / 技术|场景|特点|
|---|---|---|
|**CAN (Controller Area Network)**|汽车、工业控制|多主机总线，实时性强|
|**Modbus RTU / Modbus TCP**|工控通信|串口或以太网封装|
|**PROFINET / EtherCAT / EtherNet-IP**|工业以太网|实时工业自动化|
|**Zigbee / Thread**|IoT 无线网|低功耗、短距离|
|**LoRa / Sigfox / NB-IoT**|IoT 广域网|超低功耗、远距离、低速率|

---

## 5️ 按“是否有 MAC 地址”分类

|是否有 MAC 地址|协议举例|是否需 ARP|
|---|---|---|
|✅ 有|Ethernet、Wi-Fi、Token Ring|✔️ 需要 ARP|
|❌ 无|PPP、HDLC、Frame Relay、ATM、LTE（内部寻址机制不同）|❌ 不需要 ARP|

---

## 6️ 从“覆盖范围”维度看

|网络类型|典型协议|范围|
|---|---|---|
|**PAN（个人网）**|Bluetooth|几米|
|**LAN（局域网）**|Ethernet、Wi-Fi|房间/楼层级|
|**MAN（城域网）**|FDDI、光纤环网|城市范围|
|**WAN（广域网）**|PPP、Frame Relay、ATM、LTE、5G|全国/全球|
|**IoT（物联网网）**|Zigbee、LoRa、NB-IoT|低功耗、小数据量设备|

---

##  7️ 总结一句话

> 链路层是“**多样性最强的层**”，负责同一链路上的数据帧传输。
> 
> - 有线 LAN：Ethernet
>     
> - 无线 LAN：Wi-Fi
>     
> - 蜂窝 WAN：LTE / 5G NR
>     
> - 点对点链路：PPP / HDLC
>     
> - 工业通信：CAN / EtherCAT / Modbus
>     
> - IoT 网络：Zigbee / LoRa / NB-IoT
>     
> 
> 上层统一跑 IP → TCP/UDP → HTTP 等协议。
